
 HTML5 영역 태그.



 # div, span 태그.

div 태그.
css스타일을 지정하거나 javascript로 특정 작업을 수행하기 위해 사용된다.

span 태그.
문서의 인라인 요소를 그룹화함.

div, span 속성.
id : id를 설정하여 css, javascript에서 활용하기 쉽도록 만든다.
name : 이름을 설정하여 css, javascript에서 활용하기 쉽도록 만든다.



 # HTML5 시멘틱 구조 태그.

시멘틱 웹.
의미론적인 웹이라는 뜻이다.

기본구성.
header : 서론, 목차, 검색, 로고 등이 들어가는 곳.
nav : 내비게이션을 의미, 페이지 이동을 위한 메뉴들이 모여 있는 곳.
section : 여러 중심 내용을 감싸는 공간을 의미.
article : 콘텐츠의 내용이 들어가는 부분.
aside : 사이드에 위치하는 공간, 보조 메뉴나 광고를 배치.
footer : 저작자의 이름이나 문서의 날짜, 연락처 등이 들어감.
