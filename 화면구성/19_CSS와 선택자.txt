
 CSS와 선택자.



 # CSS, 선택자 개요.

CSS Cascading Style Sheets.
글꼴이나 배경색, 너비와 높이, 위치 등 레이아웃 지정 가능.
웹 브라우저, 스크린 크기, 장치에 따라서 화면을 다르게 표시될 수 있도록 지정 가능.

CSS 구성요소.
선택자, 스타일 속성, 스타일 값 

CSS 적용방법.
1. <p style="color:blue;"></p>
2. <head><style> p { color : blue; } </style></head>
3. <link rel="stylesheet" href="file.css">

선택자.
HTML 요소를 찾기 위해 이름, ID, 클래스, 특성 등을 지정하여 원하는 스타일 기능 적용 가능.
' , ' 로 다수의 선택자 사용 가능.

선택자 우선순위.
1순위 : p { color : blue !important; } 
2순위 : 태그안에서 적용하는 style.
3순위 : 아이디 선택자.
4순위 : 클래스, 속성, 가상선택자.
5순위 : 요소 선택자.
6순위 : 전체 선택자.



 # 선택자 종류.

선택자 종류.
전체 선택자 : *
태그 선택자 : 태그이름 
자손 선택자 : 부모태그 자식태그
클래스 선택자 : . 클래스이름
아이디 선택자 : #아이디이름
속성 선택자 : 태그[ 속성 = 속성값 ]
