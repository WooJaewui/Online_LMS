 # 웹페이지 레이아웃.

레이아웃 요소.
<details> : 추가 세부 정보 정의
<summary> : <details> 요소의 제목을 정의.

영역 분할 순서.
웹페이지 구상 -> 구성 영역 분리 -> 구성 영역 행 단위 분리 -> 나누어진 행의 내부 요소 분리.



 # 반응형 웹페이지 레이아웃.

반응형 웹 활용 시 주의사항.
유저가 사용하는 모니터의 화면 해상도 고려.
@media를 이용하여 화면 해상도에 따라 다른 레이아웃 지정.

미디어쿼리.
@media로 <style></style>에서 선언하여 미디어 유형/ 장치에 따른 스타일을 적용함.

미디어 쿼리 선언.
@media not | only mediatype and ( media feature and | or | not mediafeature ) { CSS-code ; } 
not : 전체 미디어 쿼리의 의미를 되돌림.
only : 미디어 쿼리를 지원하지 않는 브라우저 지정된 스타일을 적용하지 못하게 함.
and : 미디어 기능을 미디어 유형 또는 다른 미디어 기능과 결합.

뷰포트 meta 태그.
<meta name="viewpor" content="width=device-width, initial-scale=1">
