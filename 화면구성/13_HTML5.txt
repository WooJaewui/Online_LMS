 # 웹.

웹의 정의.
World Wide Web, WWW, W3라는 용어로 쓰임.
인터넷에 연결된 컴퓨터들을 통해 사람들이 정보를 공유할 수 있는 전 세계적인 정보 공간.

웹의 특징.
텍스트, 그림, 소리, 영상 등과 같은 멀티미디어 정보를 하이퍼텍스트 방식으로 연결.

하이퍼텍스트.
웹 상에서 존재하는 문서끼리 서로 참조( 연결 )하는 기술.

HTML Hyper Text Markup Language.
하이퍼텍스트를 사용해서 웹상에서 문서를 작성하는 방식이다.

HTTP Hyper Text Transfer Protocal.
하이퍼텍스트 문서를 서로 전공하는 방식.
누구나 검색하고 접근가능.

구성요소.
웹 클라이언트 : 필요한 데이터를 웹 서버에 요청하는 주체.
웹 브라우저 : Request Message를 작성하여 웹 서버에 전달.
HTTP : 웹상에서 정보를 주고 받을 수 있는 프로토콜. ( TCP 80 port를 사용, HTTPS 433번 포트 사용 )
웹 서버 : 클라이언트의 요청에 따라 HTML 문서를 클라이언트에게 제공해주는 주체.
웹 어플리케이션 : 브라우저 등을 통해 접근할 수 있는 응용프로그램. ( ASP, JSP, PHP, ASPX )



 # HTML5 기능.

HTML5 정의.
HTMl의 5번째 버전.
WWW의 핵심 마크업 언어.

HTML5 특징.
HTML5는 기존의 모든 문서 타입을 래핑 함.
시멘틱 마크업을 위한 의미적 요소들이 강화.
SVG 및 MathMl을 인라인으로 활용 기능 제공.
새로운 웹 폼 속성들을 이용.
플러그인 없이도 비디오와 오디오를 비롯한 미디어 재생이 가능.
