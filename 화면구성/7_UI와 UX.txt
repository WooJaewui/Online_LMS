
 UI와 UX.



 # User Inteface.

User Interface 정의.
사람들이 컴퓨터, 시스템, 기기, 도구 등 그 사이에서 일어나는 상호작용을 매개하는 것.
사용자와 사물 등 사이에서 의사소통을 위한 목적으로 만들어진 매개체.

UI 목적.
좋은 사용성 : 사용자가 필요로 하는 요소를 쉽게 찾고 사용함.
상호작용 수단과 방식 제공 : 입-출력 제공.

CLI Command Line Interface.
텍스트 터미널을 통해서 사용자와 컴퓨터가 상호작용하는 방식.
모든 입출력을 문자로 표현.

TUI Text-based User Interface.
터미널 사용자 인터페이스.
GUI와 비슷한 명령 줄 인터페이스와 다름.

GUI Graphical user Interface.
직관적인 조작방식으로 모든 명령을 알기 쉽게 그래픽으로 표현됨.

AUI Auditory User Interface.
음성 인식 등와 같이 소리를 통한 인터페이스 방식.

NUI Natural User Interface.
사람의 말과 행동으로 기기를 조작하는 방법. 
가장 직관적인 사용이 가능함.
접촉식 : 신체를 직접 기기에 접촉하여 조작. ( 터치, 뇌파 인식 등 )
비접촉식 : 접촉을 하지 않고 인식기술을 통해 조작. ( 음성 인식, 영상 인식, 시선 인식 등 )

UI 기본 원칙.
직관성 Intuitiveness : 구조를 쉽게 이해하고, 쉽게 사용할 수 있도록 제작.
유효성 Efficiency : 정확하고 완벽하게 사용자의 목표가 달성될수 있도록 제작.
학습성 Learnability : 쉽게 학습하고, 쉽게 접근하고, 쉽게 기억하도록 보장.
유연성 Flexibility : 사용자의 실수를 최대한 방지할 수 있도록 제작.

UI 설계 지침.
가시성의 원칙 Visibility : 주요기능을 노출해 최대한 조작이 쉽도록 구성.
조작결과 예측의 원칙 Natural Mapping : 조작결과를 UI만 보고도 예측 가능하도록 설계.
일관성의 원칙 Consistency : 조작방식에 일관성을 제공함으로써 쉽게 사용하도록 설계.
단순성의 원칙 Simplicity : 조작방식을 단순화시켜 인지적 부담을 줄임.
지식 배분의 원칙 Knowledge in World & Head : 쉽게 상요하도록 만듬.
조작오류의 원칙 Design for Error : 오류를 쉽게 발견될 수 있도록 하고, 수정도 쉽게 설계.
제한사항 선택사용의 원칙 Constraints : 제한사항을 명시.
표준화의 원칙 Standardization : 기능구조와 디자인을 표준화.
행동유도성의 원칙 Affordance : 조작 방법에 대한 단서를 제공.
접근성의 원칙 Accessibility : 다양한 계층의 사용자를 수용할 수 있도록 고려.



 # User eXperience.

UX 정의.
사용자가 어떤 서비스를 이용하면서 느끼고 생각하게 되는 지각과 반응, 행동 등의 경험.

UX 특징.
주관성 : 인간의 경험은 그 사람에 따라 주관적이다.
맥락성 : 사용자 경험이 일어나는 상황적 외적 환경에 영향.
총체성 : 경험 시점에서 개인이 느끼는 심리적, 감성적 결과.

UX 목표.
풍부한 사용자 경험을 제공하는 것.
양질의 사용자 경험을 제공하기 위해 사용자에 대한 깊이 있는 이해.
사용자 Needs의 도출, 욕구 충족.

UX 관점.
인지적 관점 : 인간이 어떻게 하나의 정보를 받아들여 인지하고 처리하는지.
사용자 맥락과 상황 : 외부 환경과 어떤 관계적 경험을 형성하게 되는지.
감각과 감성 : 경험 발생 시점에서의 심리적 상태와 변화의 이해.
사용자 행위 : 서비스 경험의 해석과 상호작용하여 어떻게 행동하는지

UX 중요성.
사용자에 귀속된 개념으로 사용자를 이해하고 공감하기 위한 사고방식.

사용자 정의
어떤 목표와 의도를 가지고 특정 시스템 등을 사용하여 반응하는 주체.

사용자의 이해.
어떤 욕구의 소유자 인지.
궁극적으로 원하는 것은 무엇인가. ( 기능적 측면 )

UX 프로세스.
비즈니스 전략 : 비즈니스 목표, 비전의 이해, 브랜드 전략 등 UX 리서치.
디자인 전략 수립 : 사용자분석, 요구사항 도출, 페르소나, 사용자 인터뷰 등.
UX 디자인 컨셉 : 사용자 테스트, 아이데이션, 컨셉 모델 등.
UX 디자인 개발 수행 : 스토리보드, 와이어프레임, 사용자 테스트 등.
