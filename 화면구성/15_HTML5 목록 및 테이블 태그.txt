
 HTML5 목록 및 테이블 태그.



 # 기본 목록 및 정의 목록 태그.

<li> 태그.
List Item의 약자로 ul, ol 태그의 내부에서 사용되는 태그.
들여쓰기 및 줄 바꿈 기능이 있음.

<li> value 속성.
1,2,3,.. 또는 a,b,c,d 등 넘버링을 할 수 있다.
disc, circle, square 등을 통해 구분 형태를 지정할 수 있다.

<ul> 태그
Unordered List의 약자로서 순서가 없는 태그 목록에 사용.
<li> 태그를 하위태그로 가지게 된다.

<ol> 태그
Ordered List의 약자로서 순서가 있는 태그 목록에 사용.
<li> 태그를 하위태그로 가지게 된다.
start 속성을 통해 시작 번호를 설정할 수 있다.
reversed 속성을 통해 li를 거꾸로 만들 수 있다.

<dl> 태그.
Definition List의 약자로 정의 목록을 만드는 데 사용.
<dt> <dd> 태그와 함께 사용.

<dt> 태그.
Definition Term의 약자로 용어의 제목을 나타내는 태그.
줄 바꿈 기능이 있음.

<dd> 태그.
Definition Description의 약자로 용어에 대한 설명을 나타내는 태그.
줄 바꿈 기능이 있음.



 # 테이블 태그.

테이블 태그 용어 정리.
<table> : 표 선언.
<caption> : 표 제목 선언.
<tr> : 테이블 행. Table Row.
<td> : 테이블 열. Table Data.
<th> : 테이블 헤더. Table Header. ( 굵은 글씨로 표현 )

테이블 태그 종류.
colgroup : 표의 서식 지정을 위하여 하나 이상의 열을 그룹화.
thead : 표의 머리글 내용을 그룹화하는데 사용.
tfoot : 표의 꼬리말 내용을 그룹화하는데 사용.
tbody : 표의 본문 내용을 그룹화하는데 사용.

<table>의 속성.
align : 어떻게 정렬할 것인지 결정.
border : 선들의 두께를 지정.

<td>,<tr>,<th> 태그의 속성.
rowspan : 셀이 확장되어야 하는 행의 수를 지정. ( 몇 개의 행을 차지할 것인지 )
colspan : 셀이 확장되어야 하는 열의 수를 지정. ( 몇 개의 열을 차지할 것인지 )
