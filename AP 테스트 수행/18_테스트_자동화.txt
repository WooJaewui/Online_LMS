
 테스트 자동화.



 # 테스트 자동화의 개요.

테스트 자동화.
단순, 반복적으로 수행되는 정형화된 테스트 업무를 자동화하는 기법.
기대되는 결과값을 테스트 자동화 도구가 확인해서 검증하는 방법이다.

테스트 자동화 도구.
테스트 관리, 소스 코드 리뷰 및 인스펙션, 테스트 설계 및 개발 등을 자동으로 지원하는 도구.

테스트 자동화 특징.
추적성 부여 : 설계부터 실행까지 테스트 활동 추적.
비용감소 품질 향상 : 인적, 물적 자원 절감.
재사용성 강화 : 테스트 케이스, 시나리오 재사용.

테스트 자동화 영역.
테스트 설계 지원 도구 : 테스트 설계 시에 활용하는 도구.
테스트 실행 및 로깅 지원 도구 : 테스트 실행을 하면서 나오는 결함과 테스트 상태를 관리해 주는 도구.
성능과 모니터링 도구 : 성능 및 부하 테스트 같은 시스템 테스트를 할 때 서버의 상태를 몸니터링하는 도구.
정적 테스트 지원 도구 : 애플리케이션을 실행하지 않고 소스 코드만으로 결함을 분석하는 도구.
테스트 관리 지원 도구 : 테스트 설계, 실행, 모니터링 및 정적 테스트 지원을 모두 통합해서 관리.



 # 테스트 자동화 도구.

설계단계.
명세 기반 테스트 설계 도구 
코드 기반 테스트 설계 도구.
테스트 관리 도구.

구현-테스트 단계.
정적 도구 분석.
리뷰 및 인스펙션 도구.
커버리지 측정 도구. ( 테스트를 통해 얼마나 측정되었는지 )

구현-테스트 단계.
동적 분석 도구.
성능-부하-시뮬레이션 도구.
기능 테스트 수행 도구.

테스트 관리.
테스트 관리 도구 : 테스트 관리 협업환경 지원.
인시던트 관리 도구 : 인시던드( 결함 ) 할당 및 모니터링 지원.
요구사항 관리 도구 : 요구사항 작성 및 추적성 모니터링 지원.
형상 관리 도구 : 각종 테스트웨어의 버전 관리.

테스트 설계.
테스트 설계 도구 : 테스트 케이스 설계 지원.
테스트 케이스 생성 : 자료 흐름도, 입력 도메인 분석, 랜덤 테스트. ( brute force ) 

정적 테스팅.
코드 분석 도구 : 룰 메트릭 제공.
구조 검사 도구 : 논리 그래프, 결함 체크.
데이터 분석 : 데이터 선언, 정의-링크 충돌 발견.
순서 검사 도구 : 이벤트 순서, 오순서 지적.

동적 테스팅.
단위 테스트 도구 : 단위 함수 기능 검증.
테스트 실행 도구 : 반복 테스트 자동 수행.
선으 테스팅 도구 : 가상의 부하 발생을 통한 성능-부하-스트레스 테스트 실행.
커버리지 측정 도구 : 코드 레벨 커버리지 츠겆ㅇ 지원.
동적 분석 도구 : 테스트 수행 시 런타임 결함 검출.
모니터링 도구 : 모니터링 지원.
보안 도구 : 보안 취약 결함 사항 검출.

테스트 관리 지원 도구.
결함 관리 도구 : Mantis. Bugzilla.
테스트 케이스 관리 도구 : Testlink.
형상 관리 : CVS, Subversion, GIT, Brazaar.
커뮤니케이션 도구 : MediaWiki, Dokuwiki.
통합프로젝트 지원 : Trac, nForge, Gforge.
버그 추적 : JIRA, TPMS, Director.
형상 관리 : MS Visual SourceSafe.
커뮤니케이션 : Confluence.

오픈 소스.
테스팅 프레임워크 : xUnit, TestNG, Jmock.
지속적인 통합 : CruiseControl, Hudson.
빌드 자동화 : Ant, Maven, Make.

RPA Robotic Process Automation.
인간이 하는 행위를 로봇이 흉내내면서 프로세스를 자동화하는 도구.
