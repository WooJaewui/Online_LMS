 # 통합 기능 테스트 실행.

통합 기능 테스트 확인 순서.
통합 기능 테스트 케이스 열기 : 나의 역할을 확인한다.
통합 기능 테스트 절차 확인 : 테스트 시나리오를 통해 어떤 방식으로 테스트를 수행해야 되는지 확인.

통합 테스트 실행 목적.
사용자 요구사항 및 기능 완성도의 검증.
End to End 대내-외 인터페이스 검증.
데이터 정합성 및 시스템 안정성 검증.

통합 테스트 준비 사항.
사용자 요구사항 목록 확인 : 테스트 시나리오를 어떻게 작성할지 요구사항을 보고 파악.
요구사항 추적 매트릭스 확인 : 각 단계별로 요구사항이 어떻게 처리되었는지 확인.

통합 테스트 실행 시 주의 사항.
사용자 요구사항은 통합 테스트에서 모두 테스트가 되어야 한다는 전제 하에 시나리오 확인.
자신이 수행해야 될 통합 테스트 시나리오가 기능, 시스템 중 어느 부분에 해당하는지 확인.

통합 기능 테스트 결함 등록 확인사항.
등록된 결함을 조회하여 어떠한 결함이 등록되었는지 확인.
결함 등록 프로세스를 참조하여 샘플 결함을 시험 등록.
결함의 심각도, 우선 순위 선정 기준을 다시 한 번 확인.

통합 기능 테스트 결과 등록.
통합 테스트 결과를 예상 결과와 비교하여 합격-불합격 여부를 등록.
장애 시 점검하는 시나리오가 자신에게 할당된 시나리오에 포함이 되었는지 확인.
