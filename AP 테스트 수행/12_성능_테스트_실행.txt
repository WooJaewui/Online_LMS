 # 성능 테스트 실행.

측정 항목.
거래 성공 비율, TPS,응답시간, 시스템 사용률.

시나리오.
부하증가 ( Ramp Up ) -> 부하유지 -> 부하감소 ( Ramp Down )
계단방식으로 테스트를 하면서 부하를 조절한다.

ReadyAPI.
가상사용자를 생성해서 부하를 증가시켜 테스트를 해주는 프로그램이다.



 # 성능 테스트 실행 결과 분석.

성능 테스트 실행 결과 수집.
성능 테스트 대상 서버 CPU, 메모리, 디스크 사용량 수집.
TPS 변화량, 응답시간 수집. ( 가장 중요 )

성능 테스트 실행 결과 분석.
목표 TPS 도달여부 분석.

LoadRunner.
테스트 실행 결과를 수집해주는 프로그램.

J-Meter.
OSS 도구이다.
TPS와 응답시간 관계를 잘 보여줌.
해당 업무 단위 성능 테스트 분석.
