
 고급 SW 정적 분석 - SW 배포.



 # 주요 용어 정리.

용어.
SW 정적 분석 : SW를 실행하지 않은 상태에서 분석하는 방법.
시큐어 코딩 : 보안 취약점을 유발시키는 코딩 방식을 사전에 검사하여 조치하는 행위.
프로세스 제어 : 일부 시스템 명령이 실행되거나 중요 Library가 호출되는 Case.
의도하지 않은 부호 확장 : 현재 Type의 큰 수를, 다른 Type으로 변환 시 음수값이 양수로 변환될 수 있는 상황.
정수 오버플로우 : 변수 Type에 따른 유효범위를 초과하는 상황.
umask : 운영체제에 존재하는 파일의 사용자의 접근 허용 여부를 나타내는 체계. 000 000 000.



 # 시큐어 코딩 사례 - 프로세스 제어.

프로세스 제어 개요.
특정 기능을 제공하기 위하여, 일부 시스템 명령이 실행되거나 중요 Library가 호출되는 Case.
공격자가 충분히 유추할 수 있는 Binary를 실행하는 환경.



 # 시큐어 코딩 사례 - 의도하지 않은 부호 확장.

의도하지 않은 부호 확장 개요.
현재 Type의 큰 수를, 다른 Type으로 변환 시 부호가 변환될 수 있는 상황.
큰 Data Type으로 변환하는 경우, 부호가 변경되는 Case.
음수값을 unsigned로 변환하는 경우 잘못된 부호 확장이 발생.



 # 시큐어 코딩 사례 - 정수 오버플로우.

정수 오퍼플로우 개요.
특정 변수 설정에 있어, 변수 Type에 따른 유효범위를 초과할 수 있는 Case.
입력값이 유효범위를 초과하여 오동작 가능성이 존재.

변수 Type의 설정 목적.
사용자 및 컴퓨터가 서로 동일한 정보를 처리.
정보 처리 단위 방법의 상호 일치. ( ex 부동 소수점 )
